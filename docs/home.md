# 介绍

### 关于

WeUI 是一套同微信原生视觉体验一致的基础样式库，由微信官方设计团队为微信内网页和微信小程序量身设计，令用户的使用感知更加统一。

### 特性

从 v1.0 开始采用了 BEM 命名风格，改成 `weui-aaa__bbb-bbb_ccc-ccc` 这种形式，`横杠`为连词符，`双下划线`区分 Block 和 Element，`单下划线`连接 Modifier。查看[变更说明](https://github.com/Tencent/weui/wiki/VS-v1.0.0)。

该样式库目前包含表单、基础组件、操作反馈、导航相关、搜索相关、层级规范等内容。

目前支持组件有 37 个，具体为：

- 表单：[`button`](#/button) [`表单结构`](#/form) `输入框状态` `验证码` `复选框` `单选框` `开关` `原生选择框` `模拟选择框` `文本域` [`list`](#/list) [`slider`](#/slider) [`uploader`](#/uploader)
- 基础组件：[`article`](#/article) [`badge`](#/badge) `flex` `footer` `gallery` `grid` `icons` `loading` `loadmore` `panel` `preview` `progress` `steps`
- 操作反馈：`actionsheet` `dialog` `half-screen-dialog` `msg 成功页` `msg 失败页` `picker` `toast` `toptips` 
- 导航相关：`navbar` `tabbar` 
- 搜索相关：`searchbar`
